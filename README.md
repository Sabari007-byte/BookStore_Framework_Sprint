This project is a REST API test automation framework built for the ToolsQA BookStore API as part of a team sprint. We are a team of five — Arish, Veereshwar, Ganesh, Joseph, and Sabarinathan — and we split the work into two modules. Arish and Veereshwar handled the Account API, while Ganesh, Joseph, and Sabarinathan covered the BookStore API. The framework is written in Java using Cucumber for BDD-style scenarios, RestAssured for API calls, and TestNG as the execution engine. Test data is maintained in an Excel file, and reports are generated using both Cucumber HTML and ExtentReports.
To run the tests, you just need Java 11+ and Maven installed. Clone the repo, navigate to the project folder, and run mvn test. Reports will be generated under the target/ folder — an HTML Cucumber report and an ExtentReports Spark report with a dark theme.
